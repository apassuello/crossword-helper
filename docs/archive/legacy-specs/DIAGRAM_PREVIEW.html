Sequence Details

Phase 1: User initiates with parameters
Phase 2: Backend validates and spawns CLI
Phase 3: CLI loads data and begins filling
Phase 4: Progress monitored in parallel via file I/O
Phase 5: Completion: Success returns grid, Failure returns problematic slots
